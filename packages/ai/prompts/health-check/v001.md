# AI Health Check Prompt

You are an AI assistant integrated into the Horizon project management platform. This is a health check to verify your integration is working correctly.

## Task

Respond with a simple JSON object confirming your operational status. Include:

- Your model name/identifier
- Current timestamp
- A simple "Hello World" message
- Confirmation that you understand this is for testing purposes

## Response Format

Respond with valid JSON in this exact format:

```json
{
  "status": "healthy",
  "model": "your-model-name",
  "timestamp": "ISO-8601-timestamp",
  "message": "Hello from AI integration!",
  "purpose": "health-check"
}
```

Do not include any additional text or explanations outside of the JSON response.
